# flomo-weekly-review

汇总本周已经生成的浮墨闪念卡、文章提炼卡和候选待办，形成可追溯的浮墨周整理报告。

## 适用场景

- 按项目、领域和问题归组本周闪念；
- 标出重复主题并复核闪念成熟度；
- 审计候选待办是否成功转换以及是否存在重复任务；
- 合并文章概念候选，优先提出对已有知识的增量更新；
- 为后续全库知识提炼提供本周输入。

## 核心边界

- 不执行浮墨同步；
- 默认不重新扫描全部浮墨原文；
- 重复出现只表示值得关注，不代表结论正确；
- 不自动把闪念建立为项目；
- 不自动把概念写入正式资源；
- 不自动合并、删除或关闭正式任务。

## 是否包含脚本

本技能当前不包含专用脚本。主题聚类、成熟度复核、概念合并和增量知识判断依赖 Agent 的语义理解；输入边界、审计字段和报告结构由 `SKILL.md` 约束。

## 文件结构

```text
flomo-weekly-review/
├── SKILL.md
├── README.md
└── evals/
    └── evals.json
```

## 典型触发方式

- “整理本周已经处理过的浮墨闪念和文章概念。”
- “检查本周浮墨候选待办的转换情况。”
- “把本周文章卡里的概念候选去重，优先补充已有资源。”
